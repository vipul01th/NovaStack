## 📌 Project Overview

**NovaStack** is a complete **software development service marketplace** where clients can browse, customize, and purchase professional development services including web applications, mobile apps, AI/ML solutions, and e-commerce platforms.

The platform features a **15% advance payment system**, **48-hour refund policy**, real-time order tracking, admin dashboard, and integrated chat support.

🔗 **Live Demo:** [https://novastack.me](https://novastack.me)

---

## ✨ Key Features

### 👤 Client Features
- 🔐 **User Authentication** - Sign up/Login with Email, Google, or GitHub
- 🛍️ **Service Catalog** - Browse 64+ services across 6 categories
- 🔍 **Search & Filters** - Search by name, filter by category and price
- 🛒 **Shopping Cart** - Add multiple services, remove items
- 💳 **Secure Payments** - 15% advance via Razorpay
- 📦 **Order Tracking** - Real-time status updates
- 💰 **Refund Policy** - 48-hour full refund guarantee
- 💬 **Chat Support** - Direct messaging with admin
- 👤 **Profile Management** - Update name, phone, avatar

### 👨‍💼 Admin Features
- 📊 **Dashboard** - View stats (orders, users, services, revenue)
- 📋 **Order Management** - View all orders, update status, add deliverables
- 💵 **Payment History** - Track all transactions
- 🔄 **Refund Processing** - Approve/reject refund requests
- 🛠️ **Service CRUD** - Add, edit, delete services
- 👥 **User Management** - View users and order history
- 📝 **Custom Requests** - Manage project inquiries
- 💬 **Chat System** - Reply to client messages

### 📊 Analytics Features
- 📈 **15+ Interactive Charts** - Order status, revenue trends, peak hours
- 📉 **Payment Distribution** - Advance vs Balance analysis
- 🏆 **Top Customers** - Identify highest spending clients
- ⏰ **Peak Hours Analysis** - Optimize support timing
